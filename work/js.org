* 快速人们
  JavaScript代码可以直接嵌入网页的任何地方，不过我们通常把
  JavaScript代码放到<head>中。
  #+BEGIN_SRC html
<html>
<head>
  <script>
    alert('Hello, world');
  </script>
</head>
<body>
<p>This is a test</p>
</body>
</html>
  #+END_SRC
** 如何运行JavaScript
   要让浏览器运行JavaScript，必须先有一个HTML页面，在HTML页面中引入JavaScript，
   然后，让浏览器加载该HTML页面，就可以执行JavaScript代码了。
